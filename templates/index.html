Guide d'utilisation
Comment configurer et utiliser le bot SMS
📱 Étape 1 — Installer l'app sur ton téléphone

Le bot envoie les SMS depuis ton téléphone Android via l'application SMS Gate. Tu dois l'installer sur chaque téléphone que tu veux utiliser.

▶ Télécharger sur Google Play 🌐 Site officiel
⚠️ REQUIS
  • Téléphone Android uniquement (pas iOS)
  • Une carte SIM active avec forfait SMS inclus
  • L'app doit rester ouverte en arrière-plan
  • Désactiver l'optimisation de batterie pour l'app
🔑 Étape 2 — Créer un compte SMS Gate

Dans l'application SMS Gate sur ton téléphone :

1
Ouvre l'app → appuie sur « Se connecter » ou « Créer un compte »
2
Crée un compte avec un email et un mot de passe — note bien ces identifiants, tu en auras besoin dans le panel
3
Une fois connecté, l'app affiche un Device ID (ex: android-xxxx-xxxx) — note-le
📡 Étape 3 — Connecter le téléphone au panel

Va dans l'onglet Devices et clique « + Ajouter ». Remplis :

Nom du device
Un nom pour identifier ton téléphone, ex: Samsung-S22
Device ID
Le code affiché dans l'app SMS Gate, ex: android-xxxx-xxxx
Username / Password
Les identifiants de ton compte SMS Gate (email + mot de passe créé dans l'app)
📋 Étape 4 — Préparer les contacts

Va dans Contacts et uploade un fichier CSV.

FORMAT DU FICHIER CSV
phone
+15141234567
+15139876543
+14381234567
  • La première ligne doit être exactement phone
  • Un numéro par ligne, avec indicatif pays
  • Format recommandé : +1XXXXXXXXXX
  • Chaque numéro est supprimé du CSV après envoi réussi (anti-doublon automatique)
✍️ Étape 5 — Rédiger le message

Tu as deux options :

Message unique (onglet Message)
Un seul texte envoyé à tous les contacts. Simple et rapide.
Templates (recommandé — onglet Templates)
Ajoute plusieurs variantes. Le bot en choisit une aléatoirement à chaque envoi — réduit drastiquement les blocages opérateurs.
🚀 Étape 6 — Lancer le bot

Une fois devices + contacts + message prêts :

Lance la campagne — le bot commence à envoyer les SMS
Met le bot en pause — les envois en cours se terminent
Reprend après une pause
Arrête complètement la campagne

Surveille les stats en temps réel sur le Dashboard et l'historique détaillé dans Historique.

❓ Problèmes fréquents
📵 Le device apparaît inactif / déconnecté ▾
Vérifie que l'app SMS Gate est ouverte sur le téléphone. Va dans paramètres Android → Apps → SMS Gate → Batterie → désactive l'optimisation. Ferme et relance l'app si le problème persiste.
❌ Les SMS échouent tous (statut FAIL) ▾
Vérifie que le Device ID, l'username et le password correspondent exactement à ceux de l'app SMS Gate. Teste en envoyant un SMS manuellement depuis l'app. Certains opérateurs bloquent les SMS en masse — augmente les pauses dans Réglages.
📵 Les destinataires ne reçoivent pas les SMS ▾
Si le statut dit "Sent" mais les gens ne reçoivent rien, l'opérateur filtre les SMS. Solutions : utilise les Templates pour varier le texte, augmente le délai (15s+ recommandé), évite d'envoyer des centaines de SMS par heure depuis le même numéro.
📂 Mon CSV ne s'uploade pas ▾
Vérifie que le fichier est au format .csv et que la première ligne est exactement phone. Pas d'espaces, pas d'autres colonnes. Si tu utilises Excel, enregistre en « CSV UTF-8 ».
🔄 Le bot s'arrête tout seul ▾
Le bot s'arrête quand il n'y a plus de contacts dans le CSV ou quand aucun device actif n'est disponible. Vérifie dans Contacts combien de numéros restent, et dans Devices que tes appareils sont activés (bouton ON).
🔐 J'ai oublié mon mot de passe ▾
Contacte l'administrateur du panel via l'onglet Support — il peut réinitialiser ton mot de passe depuis le panneau Utilisateurs.
